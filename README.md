# WiApIn

Приложение для установки программ, инструкции для винды, а также скрипты

В первую очередь эта штука делалась для учеников академии штиглица, так как будет проще, чтобы все можно было посмотреть и скачать самим, а не носить мне ноуты.

Делалась данная работа через ИИ "Gemini", ибо опыта в написание десктопных приложений у меня пока не было.

Мой репозиторий, откуда берутся приложения/архивы для установки: https://huggingface.co/datasets/TheMuramasa/WiApIn-Apps/tree/main

# Установка

Для установки выберите справа раздел "Releases":

![ТЕСТТЕСТ](https://raw.githubusercontent.com/farhhh/WiApIn/main/Instructions/images/releases.png)

На открывшейся странице выберите последнюю версию приложения (она всегда сверху) и нажмите под версией на файл с расширением .exe:

![ТЕСТТЕСТ](https://raw.githubusercontent.com/farhhh/WiApIn/main/Instructions/images/download.png)

Дождитесь установки. Если на файл ругается антивирус - выберите "установить в любом случае". Ругается он из-за некоторых разрешений приложения, это нормально.

Далее прочитайте вводную инструкцию в самом приложении.

# Как работает

Приложение парсит этот гитхаб, а также репозиторий хаггинг фейса, для того, чтобы подгружать приложения и архивы.

Поэтому обновление приложения не обязательно, т.к. оно будет подхватывать новые инструкции, скрипты и приложения прямо отсюда.
