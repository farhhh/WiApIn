# WiApIn

Приложение для установки программ, инструкции для винды, а также скрипты

В первую очередь эта штука делалась для учеников академии штиглица, так как будет проще, чтобы все можно было посмотреть и скачать самим, а не носить мне ноуты.

Делалась данная работа через ИИ "Gemini", ибо опыта в написание десктопных приложений у меня пока не было.
